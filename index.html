Raspberry Pi Cheat Sheet

Raspberry Pi Cheat Sheet

Category Command/Configuration Description
Update System sudo apt update Updates the package list.
sudo apt upgrade Upgrades installed packages to the latest versions.
Install Software sudo apt install <package_name> Installs a specific package. Replace <package_name> with the desired software.
File Management ls Lists files in the current directory.
cd <directory> Changes the current directory to <directory>.
cp <source> <destination> Copies files from source to destination.
mv <source> <destination> Moves or renames files.
rm <file> Deletes a file.
Networking ifconfig Displays network configuration.
ping <address> Tests connectivity to a specific address.
ssh <user>@<ip_address> Connects to another machine via SSH.
Raspberry Pi Config sudo raspi-config Opens the Raspberry Pi configuration tool.
Reboot/Shutdown sudo reboot Reboots the Raspberry Pi.
sudo shutdown now Shuts down the Raspberry Pi immediately.
Python python3 <script.py> Runs a Python script.
GPIO Contro import RPi.GPIO as GPIO Imports the GPIO library in Python.
GPIO.setmode(GPIO.BCM) Sets the pin numbering system to BCM.
GPIO.setup(<pin>, GPIO.OUT) Configures a pin as an output.
